# PowerStripHolder
A 3D printed mount for a multiple socket power strip for attachment to the edge of a table

![Top view](assets/images/PowerStripHolder.jpg)

![3d print parts](assets/images/3dPrintParts.jpg)

![3d print assembled parts](assets/images/3dPrintPartsAssembled.jpg)

| Property  | Value |
| ------------- | ------------- |
| Tabletop height | 28mm |
| Powerstrip width | 46mm |
| 3D print weight | 105g |

## Appendix

### STL

- [Holder](STL/PowerStripHolder.stl) (Two prints are needed)
- [Insert](STL/PowerStripInsert.stl) (Two prints are needed)

All STL files were created and designed with Tinkercad https://www.tinkercad.com

Printer settings used on my Anycubic Vyper (Thank you **https://github.com/m-holler**!): 
- Nozzle: 0.4mm
- Layer height: 0.15mm
- Filament: PLA+ silk
- Infill density: 25%
- Raft layers: 1 layer (Optional, for the insert to prevent warping)
